Pertahankan design system aplikasi KebunKu berikut:

Style utama:
- Modern agriculture dashboard
- Clean, rounded, soft shadow
- Nuansa pertanian premium
- Mobile friendly dan responsive
- Layout dashboard dengan sidebar kiri dan topbar

Warna utama:
- Background utama: #F5F7F2
- Sidebar: #123D2A
- Primary green: green-700 / emerald-600
- Accent lime: lime-400
- Text gelap: green-950 / gray-900
- Text soft: green-100 / green-200 / gray-500
- Card background: white atau green-50
- Border: green-100 atau white/10

Komponen wajib:
- Sidebar gelap hijau dengan menu grouped section
- Active menu putih dengan text green-950
- Icon active bg-lime-100 text-green-700
- Icon inactive bg-white/10 text-green-100
- Header sticky bg-white/80 backdrop-blur-xl
- Hero/page header gradient from-green-700 via-emerald-600 to-lime-600
- Semua card rounded-2xl atau rounded-3xl
- Button utama bg-green-700 hover:bg-green-800
- Button secondary border-green-100 bg-white
- Badge active bg-green-100 text-green-700
- Badge inactive bg-gray-100 text-gray-600
- Table modern dengan rounded-xl, shadow-sm, border-green-100
- Mobile table berubah menjadi card list

Jangan ubah identitas visual ini.
Jangan gunakan warna biru sebagai warna utama.
Jangan gunakan layout kotak tajam.
Jangan gunakan sidebar putih.
Gunakan Tailwind CSS.
Gunakan React + Inertia.js.
